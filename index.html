Some useful Info   


By the default, a socks parsing config looks like follows:

The SNI (Server Name Indication) value is set to 'Others...' . The appropriate field of the request is empty. You can easily change this field choosing pre-installed values from the drop-down list or fill it in with your exclusive value. Leaving this field empty shall allow you to try to use a sock's Owner SNI if it is exists in the original configuration.

The Country's filter is turned to the most popular traffic exchange points & hosting data centers in Europe located in the Great Britain, Germany, Sweden, France and the Holland. You can edit this field manually, adding/removing a Country Codes. The Codes MUST follow one by one divided by semicolons. Leaving this field emty you'll get a socks from ALL the Countries currently stored in a database.

The VPN Protocols are not selected, thus it means NO FILTERS are applied (this is equlal to all checkboxes are checked i.e. all the Protocols are selected).

The Security Filter is set to the 'tls' value and the Port value is equal to 443. You can also change those values manually. Each value in those fields must follow by semicolon, as well. You can delete these values and send an unconfigured 'Security filter' request. In this case you can get a mixed 'cloud of socks' with a various Security config values, as well as socks without Security configs. Any case, you have to have in your mind that sock's Owner may follow your online activity and collect some of your Personal data. So the usage of a socks is fully your Private Desicion! Never the less, to prevent a possible data leaks its recommended to use a so called 'sock chain' finished with WARP/WARP+ config.*

As fare as you created desired configuration (or leaved the default one), click/tap on the 'GET'EM NOW' button. Then select and copy to the clipboard everything you want from a pop-up window. If you got an empty window, return into the main page and try to create a new config, repeat your request and in a case of successful result come through the select-copy procedure. Then paste a copied data into a proper app, for ex. into the NekoBox App or its followers/clones.

To obtain the latest version of this application simply long click/tap on the Socks app icon (CheburGoog) located on your desktop and DELETE ALL APP DATA in the 'App. info' item of your Android device. Keep cleaning this app data from time to time to always stay updated.
